# JavaScript Mini Projects
A collection of small, functional JavaScript projects designed to practice DOM manipulation, logic building, and modern ES6+ features.

مجموعه‌ای از پروژه‌های کوچک و کاربردی جاوا اسکریپت برای تقویت مهارت‌های منطقی و کار با DOM.

<br>

## 📋 Project list


### 1. Interactive Calculator
A sleek calculator capable of basic arithmetic operations, remainder calculations, and exponentiation.
- **Key Features:** Keyboard support simulations, double-click events, and clean UI.
- **Tech Used:** `getElementById`, Event Listeners (`click`, `dblclick`), `slice()`, `eval()`.

<br>

### 2. Inventory Management System
A dynamic system to manage product listings, calculate total stock value, and filter out-of-stock items.
- **Key Features:** Search by name, out-of-stock filtering, and total value calculation.
- **Tech Used:** Array Methods (`filter`, `forEach`), Objects, `querySelector`, `innerHTML`.

<img width="1593" height="754" alt="Screenshot" src="https://github.com/user-attachments/assets/717db510-a0e1-43e0-a5b7-bded6e5142a7" />
<br>

### 3. Discount & Tax Calculator
A utility tool that calculates the final price of a product after applying user-defined discounts and a fixed 10% tax.
- **Key Features:** Error handling for invalid inputs and real-time calculation.
- **Tech Used:** `let/const`, `if-else` logic, `try-catch` blocks, Functions.

## 💡 Learning Objectives
The main goal of this repository is to demonstrate proficiency in:
- **Event Handling:** Capturing and responding to user interactions.
- **Data Structures:** Efficiently using Arrays and Objects.
- **DOM Manipulation:** Dynamically updating the UI based on data.
- **Logic & Math:** Implementing real-world formulas in code.
